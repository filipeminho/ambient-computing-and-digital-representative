# Ambient Computing and Digital Representative Negotiation

This repository contains materials related to the paper:

**The New Frontier – The Role of Ambient Computing in Digital Representative Negotiation**  
Wei Kong, Kevin Dragomir, Filipe Minho, and Calvin Lim  
Saint Mary’s College of California  
December 2019

## Overview

This project explores the role of ambient computing in digital representative negotiation, with a focus on resolving preference-based conflicts in shared environments. The paper examines how sensor-based systems, behavioral science, optimization, and decision tree models can be used to support negotiation processes, particularly in the context of thermal comfort.

## Repository Contents

- `paper/` — paper files and supporting materials
- `code/` — Python code used in the analysis and modeling
- `figures/` — charts, visualizations, and generated outputs
- `data/` — datasets or sample inputs used for the project, if included

## Purpose

This repository is shared for research, review, and archival purposes in connection with the paper and its submission materials.
